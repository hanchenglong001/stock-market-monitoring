# 股票盯盘服务

## 项目简介

本项目是一个股票盯盘服务，使用新浪行情接口获取股票实时行情数据，通过同花顺下单接口实现股票交易操作。该服务用于实时监控股票行情，并在满足特定条件时自动执行下单操作。

### 技术栈

- **行情接口**：新浪（开源 API）
- **开发语言**：Python 3.9
